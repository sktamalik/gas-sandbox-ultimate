# 🚀 GAS Sandbox (Modern Ultimate v5.0)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-5.0.0-blue.svg)](https://github.com/)

**GAS Sandbox** adalah alat simulasi lingkungan _Google Apps Script_ (GAS) berbasis web. Alat ini memungkinkan pengembang untuk menguji tampilan HTML frontend dan mensimulasikan pemanggilan fungsi backend (.gs) tanpa harus berpindah-pindah tab ke Google Script Editor asli.

---

## ✨ Fitur Utama

- **Dual Editor:** Editor kode HTML dan JavaScript (Backend Mock) yang terpisah secara rapi.
- **Smart Auto-Generate:** Menganalisis kode HTML secara mendalam untuk membuat fungsi dummy backend secara otomatis.
- **Ultra Sniffer 2D Array:** Deteksi otomatis struktur data, baik berupa objek maupun Array 2D.
- **Real-time Preview:** Jalankan dan lihat hasil perubahan kodemu secara instan.
- **Modern UI:** Antarmuka gelap (_Dark Mode_) yang elegan dengan fitur _resizer_ yang fleksibel.

## 🛠️ Teknologi yang Digunakan

- **Frontend:** HTML5, CSS3 (Bootstrap 5, FontAwesome 6)
- **Editor:** [CodeMirror](https://codemirror.net/) untuk pewarnaan sintaksis.
- **Logic:** Vanilla JavaScript dengan sistem _Proxy_ untuk mensimulasikan `google.script.run`.
